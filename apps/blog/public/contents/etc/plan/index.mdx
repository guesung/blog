#### 회사 내
회사를 다니는 동안에는 무조건 회사 업무가 1순위이다. 만약, 기한 내에 회사 업무를 마치지 못했다면, 퇴근 후 혹은 주말을 이용해서 수행하도록 하자.
- **목표**
    1. '저 이거 했어요'할 만한 실적 쌓기
    2. 무조건 적극적으로 나서서 업무 다양하게 수행하기
- **계획**
    1. 주어진 프로젝트 성공적으로 마무리하기
        1. AIDT 교과서 개발 (70% 완료)
        2. 모노 레포 (70% 완료)
        3. 사내 블로그 개발 (0%)
        4. 랜딩 페이지 성능 개선 (5%)

#### 회사 외
- **목표**
    1. 개발 블로그 일상화하기
    2. 기술에 대해 깊이 있게 아는 개발자 되기
- **계획**
    1. 블로그 1주일 1포스팅 실천하기
    2. 블로그 포스팅 후, 링크드인에 홍보하기
    3. 기술 스택 하나씩 딥다이브하기 : `Node.js`, `React.js`,`Next.js`
